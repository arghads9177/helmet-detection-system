# Helmet Compliance Detection (POC)

Monitor an entry point using a CCTV camera and automatically detect whether people entering the monitored area are wearing safety helmets.

The system should:

- Allow normal entry for people wearing helmets.
- Generate an alert only when one or more people enter without wearing helmets.
- Capture evidence snapshots.
- Record timestamps.
- Generate event logs.
- Display alerts and event history through a web dashboard.

This POC validates the complete workflow before implementing the production-grade helmet compliance monitoring system.
